# AI Engineering Manager | エンジニア成長支援システム

このプロジェクトは、AIがあなたの「上司（エンジニアリングマネージャー）」として機能し、技術スキルの自己診断からキャリアロードマップの策定、週次の課題アサイン、そして高度なコードレビューまでを一貫してサポートする成長支援システムです。

---

## 🚀 はじめに

エンジニアとしての成長を加速させるためには、「適切な目標設定」「継続的なアウトプット」「質の高いフィードバック」の3要素が不可欠です。本アプリは、Google Gemini 2.0 Flash を活用し、これらすべてを自動化・高度化します。

## 📖 取扱説明書 (User Manual)

### 1. スキル診断とオンボーディング
アプリを起動したら、まずは「スキル診断」から始めましょう。
- **診断の流れ**: 現在のロール（職種）、経験年数、目標とするキャリア（例：テックリード、フルスタック等）を入力し、主要な技術カテゴリ（フロントエンド、バックエンド、インフラ等）の自己評価を行います。
- **AIによる分析**: 入力されたデータに基づき、AIがあなたの現在の立ち位置を客観的に分析し、初期プロファイルを生成します。

### 2. キャリアロードマップの策定
診断が完了すると、自動的に「36ヶ月の成長ロードマップ」が生成されます。
- **段階的なステップ**: 「直近3ヶ月」「6ヶ月」「12ヶ月」といったタイムスパンで、習得すべき技術や役割の変化が明示されます。
- **パーソナライズ**: あなたの理想のキャリア像に合わせて、学習の優先順位が最適化されます。

### 3. 週次課題 (Challenges)
ロードマップを達成するために、毎週AIが具体的な「課題」をアサインします。
- **課題の構成**: 課題は複数の「受け入れ条件（PR単位のタスク）」に分割されています。
- **設計レベルの分割**: 課題は「DB設計」「API・Repository実装」「UI開発」のように、実際の開発現場に近い粒度で細分化されています。
- **獲得ポイント**: 課題を完了することで、対応する技術カテゴリのスキルポイントを獲得し、プロフィールのレベルが上昇します。

### 4. 高度なAIコードレビュー (Gemini 2.0 Flash)
実装したコードをAI上司がレビューします。本アプリの最も核となる機能です。
- **レビューの依頼**: 「AIレビュー」画面から、紐づける課題と対象のPR（受け入れ条件）を選択し、コードを貼り付けて送信します。
- **5軸評価**: 「設計」「命名」「エラーハンドリング」「テスト」「パフォーマンス」の5つの観点でスコアリングされ、レーダーチャートで表示されます。
- **ソクラテス式フィードバック**: AIは単に答えを教えるのではなく、「なぜこのアプローチを選択したのか？」といった問いかけを通じて、あなたの思考を深めます。
- **自動進行**: レビューで「Approved（承認）」が得られると、対象の受け入れ条件は自動的に完了扱いになります。すべての条件をクリアすると課題達成となります。

### 5. 成長の可視化とダッシュボード
あなたの成長は常に可視化されます。
- **キャリアハイライト**: 獲得した総ポイントや達成した課題数が表示されます。
- **強みと伸び代**: AIレビューの結果から蓄積された「あなたの現在の強み」と、次のレベルに進むために必要な「今後の伸び代（弱点）」がリストアップされます。
- **月次レポート**: 1ヶ月の活動ログをAIが総括し、フィードバックと次の月の注力ポイントを提示します。

## 🛠 技術スタック

- **Frontend**: Next.js 14 (App Router), TypeScript, Tailwind CSS
- **State Management**: LocalStorage persistence (Custom Hook based)
- **AI Engine**: Google Gemini 2.0 Flash API (gemini-2.0-flash-exp)
- **Iconography**: Lucide React
- **Charts**: Chart.js / react-chartjs-2

## 📦 セットアップ

1. **環境変数の設定**:
   `.env.local` ファイルを作成し、Gemini APIキーを設定してください。
   ```bash
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

2. **依存関係のインストール**:
   ```bash
   npm install
   ```

3. **開発サーバーの起動**:
   ```bash
   npm run dev
   ```

## 📐 デザインコンセプト

- **Aesthetics**: 深みのあるダークテーマを採用し、ガラスモフィズム（Glassmorphism）を活用したプレミアムな質感。
- **Responsiveness**: ノートPCからモバイルデバイスまで対応。デスクトップではサイドバー、モバイルではボトムナビゲーションによる快適な操作性を提供します。
